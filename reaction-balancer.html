The Math Behind the Balancer

For a reaction like $aA + bB \rightarrow cC + dD$, we build a matrix where each row represents an element (C, H, O, etc.). We solve for the vector $\vec{x} = [a, b, c, d]^T$ such that: $$ \text{Matrix} \cdot \vec{x} = 0 $$ The coefficients must be the smallest integers, obtained by finding the **Least Common Multiple** of the denominators in the Reduced Row Echelon Form.
